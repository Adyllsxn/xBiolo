# ❓ Perguntas frequentes (FAQ)

## Para clientes

### Preciso criar conta para comprar?
Não. Você só preenche nome e endereço na hora de finalizar o pedido.

### Como pago?
A forma de pagamento é combinada direto com o lojista no WhatsApp. Pode ser dinheiro, transferência, cartão, etc.

### Como recebo o produto?
Combinado com o lojista. Pode ser entrega em casa ou retirada.

### O Biolo é seguro?
O Biolo só mostra os produtos. A venda é entre você e o lojista direto no WhatsApp.

### Preciso instalar algum aplicativo?
Não. O Biolo funciona no navegador do celular ou computador.

---

## Para lojistas

### Quanto custa?
Tem plano grátis (Biolo Borla) e planos pagos a partir de 2.500 Kz/mês.

### Como recebo o dinheiro das vendas?
O cliente paga direto pra você. O Biolo não recebe o dinheiro.

### Preciso de site próprio?
Não. O Biolo já te dá o link do catálogo.

### Posso usar meu próprio WhatsApp?
Sim. O pedido vai direto pro seu WhatsApp.

### Quantos produtos posso cadastrar?
No plano Borla: até 10 produtos. Nos planos pagos: ilimitados.

### Como faço para ter várias lojas?
Isso é no plano Biolo Salu. Cada vendedor tem seu próprio WhatsApp.

### Preciso saber programação?
Não. O dashboard é simples, igual usar redes sociais.

### Tem suporte?
Sim. Plano Raço e Salu têm suporte prioritário pelo WhatsApp.
