# 🛍️ Como funciona para o cliente

## O que é o Biolo?

Biolo é um catálogo digital. O lojista mostra os produtos dele e você escolhe o que quer comprar.

---

## Passo a passo

### 1. Receber o link

O lojista te manda um link pelo WhatsApp. Exemplo: `https://biolo.ao/loja/seunome`

### 2. Escolher os produtos

Dentro do catálogo você pode:
- Ver todos os produtos
- Filtrar por categoria
- Filtrar por preço
- Clicar no produto pra ver detalhes

### 3. Botar na sacolinha

Em cada produto você:
- Escolhe o tamanho (P, M, G, etc)
- Clica em "botar na sacolinha"

### 4. Ver a sacolinha

Clica no botão "Sacolinha" pra ver tudo que você escolheu.

Pode:
- Aumentar ou diminuir quantidade
- Remover produtos
- Ver o total

### 5. Fechar o pedido

Clica em "Fechar pedido" e preenche:
- Seu nome
- Seu endereço
- Forma de pagamento (Pix, cartão, dinheiro)

### 6. Enviar

Clica em "Enviar pedido". O WhatsApp abre com o pedido pronto.

Você só precisa apertar "Enviar".

### 7. Receber a confirmação

O lojista vai te responder no WhatsApp combinando:
- Entrega
- Pagamento
- Prazo

## Não precisa de cadastro

Diferente de outros sites, você **não precisa criar conta** no Biolo. Só preenche nome e endereço na hora de comprar.
