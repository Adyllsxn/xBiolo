# 👨‍💼 Como funciona para o lojista

## O que você precisa para começar

- Um celular com WhatsApp
- Produtos para vender
- Vontade de vender mais

---

## Passo a passo

### 1. Cadastro

Você cria sua conta no Biolo com seu nome e WhatsApp.

### 2. Cadastrar produtos

No dashboard do lojista, você adiciona:
- Nome do produto
- Categoria (vestido, blusa, calça, etc)
- Descrição
- Preço em Kz
- Variações (tamanhos P, M, G, etc)
- Fotos (em breve)

### 3. Compartilhar o catálogo

Você recebe um link do seu catálogo. Exemplo: `https://biolo.ao/loja/seunome`

Compartilha esse link no:
- WhatsApp (status, grupos, contatos)
- Instagram
- Facebook
- Ou manda direto pro cliente

### 4. Receber pedidos

O cliente entra no seu catálogo, escolhe os produtos e finaliza.

O pedido chega no seu WhatsApp com:
- Nome do cliente
- Endereço
- Itens escolhidos
- Total a pagar
- Forma de pagamento escolhida

### 5. Vender

Você responde o cliente no WhatsApp e combina:
- Entrega
- Pagamento
- Prazo

---

## O que você pode fazer no dashboard

| Funcionalidade | O que faz |
|---|---|
| **Ver produtos** | Lista todos os seus produtos |
| **Adicionar produto** | Cadastra um novo produto |
| **Editar produto** | Muda preço, descrição, etc |
| **Desativar produto** | Produto some do catálogo |
| **Ver pedidos** | Histórico de pedidos |
| **Configurar loja** | Muda seu WhatsApp, nome da loja, etc |

## Dicas para vender mais

- **Produtos com foto vendem mais** (em breve vai ter upload)
- **Preços claros** evitam perguntas desnecessárias
- **Responda rápido** no WhatsApp depois do pedido
- **Compartilhe o link toda semana** nos status
